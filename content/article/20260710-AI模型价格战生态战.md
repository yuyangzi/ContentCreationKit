# AI 模型价格战本质是生态战：GPT-5.6、Grok 4.5、Muse Spark 同日出牌

> 7 月 9 日，24 小时内三场 AI 模型发布接连落地——GPT-5.6 全量上线、Grok 4.5 突袭、Muse Spark 1.1 亮剑。三家不约而同把价格压到同一区间，媒体标题清一色是"价格战""打骨折"。但降的不是价，是入场券。

---

7 月 8 日下午，SpaceXAI 扔出 Grok 4.5。马斯克说它"大致对标 Opus 4.7"，输出定价每百万 token 6 美元（SpaceXAI, 2026-07-08）。

不到 24 小时后，OpenAI 全量上线 GPT-5.6。三档定价里最低的 Luna 输出也是 6 美元（OpenAI, 2026-07-09）。Meta 紧跟着发布了 Muse Spark 1.1——扎克伯格三年来的第一条推文——输出价 4.25 美元（Meta, 2026-07-09），比 Luna 还低 29%。

三场发布，同一价格区间。媒体标题很整齐："AI 模型白菜价时代来了。"

但 Claude Fable 5 仍然标着 50 美元。

---

如果行业真的在打价格战，为什么 Fable 5 纹丝不动？为什么 Opus 4.8 稳在 25 美元？价格战只发生在一个非常具体的区间——输出价 5 美元以下的"够用就行"模型——而高价模型的定价逻辑在很大程度上与此无关。

市场在分叉，不是在统一降价。

低端这一侧，Luna（$6）、Grok 4.5（$6）、Muse Spark（$4.25）在血拼。再往下看，DeepSeek V4 Pro 输出价 0.87 美元（DeepSeek API Pricing），不到美国模型的五分之一。

高端那一侧呢？Fable 5 在 6 月 12 日至 7 月 1 日间因出口管制暂停，恢复后输出价仍标着 50 美元（Anthropic Platform Pricing），Anthropic 没有任何降价的迹象。买 Fable 5 的人不是为了省钱——他们买的是"一次搞定"，是 SWE-Bench Pro 上 80.3% 的准确率，是 OpenAI 自建的 Agent 末考里被 GPT-5.6 Sol（53.6%）大幅领先的那个 40.5%。

这里的分叉不是两个价格段的分叉，是两种需求的分叉。一边要够用，一边要最好。

为什么 GPT-5.6 Sol 在 Agent 末考上跑出 53.6%，输出价却只有 30 美元——仅为 Fable 5 的 60%？定价策略反映的不是成本，而是战略——OpenAI 并非不能定高价，只是选择了不这么做。

---

三家公司降价的同时，各自动了同一件事：工具链。

OpenAI 把 Codex 并入了 ChatGPT Work。Codex 这个独立产品不存在了，变成了 ChatGPT 里的一个 Agent 能力。"Sites"开始公测——你可以在 ChatGPT 里搭建和部署一个网站，不需要离开这个界面。Atlas 浏览器也关了。OpenAI 在收拢生态，方向很明确：从卖模型变成卖一个你不需要离开的平台。

再看 SpaceXAI。今年 6 月以 600 亿美元收购了 Cursor。Grok 4.5 是收购后第一款联合训练的模型。一个模型公司和一家 IDE 绑在一起——Grok 在 Cursor 里的表现直接决定了开发者会不会留下来。降价是在给 Cursor 引流，不是在做慈善。

Meta 的动作更微妙。今年 6 月，Meta 内部开始限制 Claude Code 和 Codex 的使用权限。然后自建了 MetaCode。然后发布了 Muse Spark——Meta 的第一个闭源收费 API。扎克伯格为此发了三年来的第一条推文。一个开源信仰者发布了一个闭源模型，还为此打破了三年的推特沉默。这不是产品发布，是战略转向。

三家在做的，本质上是一件事：用低价模型把你拉进自己的工具链，然后靠生态效应把你锁住。Token 价格只是获客成本。

---

DeepSeek 不在发布名单里，但它的影子一直在。

V4 Pro 输出价 0.87 美元（DeepSeek API Pricing），SWE-bench Verified 上自己报了 80.6%（DeepSeek, 2026-07）。这个价格不到 Opus 4.8 的 1/28，不到 GPT-5.6 Luna 的 1/7。OpenRouter 上有 51% 的 API 流量走的是中国模型——不是美国用户不想要便宜的模型，是出口管制把 DeepSeek 挡在了外面。

于是出现了一个悖论：美国公司这一轮降价，真正害怕的对手不在硅谷——但那个对手又进不来。DeepSeek 是一个"近在咫尺又远在天边"的竞争者：它把全球定价锚点压到了地板，但它的模型在美国市场不可用。这给了 OpenAI 和 SpaceXAI 一个窗口期——在出口管制的保护伞下，用低价先把美国开发者锁住。一旦管制松动，价格战就不是 $6 vs $4.25 了，是 $6 vs $0.87。

这个窗口有多长，主要取决于华盛顿，而非硅谷。

---

GPT-5.6 的三档定价（Sol $30 / Terra $15 / Luna $6），是一次清晰的市场分层——用 Sol 打高端，用 Luna 打低端，用 Terra 填中间。Grok 4.5 用"Opus 4.7 级，打骨折价"的叙事，试图在开放权重和 Cursor 生态之间卡一个位。Muse Spark 用最低价和最晚入场的身份，赌的是 Meta 的工程能力和内部需求能给它持续的迭代动力。

但定价只是表层竞争。

同一天，OpenAI 关了一个浏览器、并了一个 IDE、开了一个建站工具。SpaceXAI 把模型和一个 600 亿美元的 IDE 焊在一起。Meta 把自己从开源信仰者的位置上挪开了。

一家公司能不能守住开发者，很可能不取决于这个月谁的 token 便宜 1 美元，而是取决于明年这个时候，开发者发现自己已经换不掉这个工具链了。

而 Anthropic 在这张牌桌上，打的是另一副牌。它不降价，它把 Claude Code 做得足够好用，把 Fable 5 的推理做得足够深，让你觉得 50 美元值这个价。这条路能不能走通，取决于一件事：当 Sol 的 Agent 末考分数已经超过 Fable 5 的时候，"更好"还撑不撑得住"更贵"。

Fable 5 的暂停也许不完全是需求太大。也可能是 Anthropic 在重新算账——当一个 30 美元的模型开始在某些指标上超过你，继续守着 50 美元的牌桌，还坐得住吗？
