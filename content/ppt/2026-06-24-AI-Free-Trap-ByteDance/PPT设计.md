# PPT设计：豆包烧钱、Seedance赚钱，字节AI的两个世界

## 1. 配色预设

**选择：预设A · 霓虹紫（默认）**

适用场景：前沿科技、AI、赛博朋克话题 → 符合字节AI主题

```css
:root {
  --nc-accent:  #a855f7;   /* 紫 → 中性强调、主标题高亮 */
  --nc-success: #22d3ee;   /* 青 → 正面数据、上升/默会知识 */
  --nc-danger:  #f43f5e;   /* 玫红 → 负面数据、下降/警示 */
  --nc-warning: #fbbf24;   /* 琥珀 → 保留 */
  --nc-info:    #818cf8;   /* 靛蓝 → 保留 */
}
```

### 颜色编码规则（各幻灯片严格遵循）

| 数据类型 | 颜色 | 使用场景 |
|----------|------|----------|
| 增长/正面 | 青色 `nc-text-success` | Seedance营收增长、付费用户提效感知 |
| 下降/负面 | 玫红 `nc-text-danger` | MAU下跌610万、净利润降70%、成本 |
| 中性/强调 | 紫色 `nc-text-accent` | DAU 2亿、Token 180万亿、各类百分比 |
| 默会知识/价值判断 | 青色 `nc-text-success` | "降本之后才能创造价值"、"免费是一个阶段" |

> 红色严格仅用于负面数据。93% AI使用率等中性数据不能标红。

## 2. 动画策略

**选择：淡入降级（B站录屏）**

- 全局 `transition: fade`
- `<style>` 中三段降级 CSS 禁用 staggered/shimmer/particles
- `<v-clicks>` 保留，仅淡入效果
