## 热度背景分析

2026年6月，企业支出管理平台Ramp发布最新AI Index，样本覆盖超7万家美国企业和数十亿美元企业支出。核心发现足以写入AI经济学教材：

**三个反直觉数据**：
- Token价格降了98%（GPT-4级别：2022年末每百万token约20美元 → 2026年约0.40美元）
- 企业AI总账单涨了320%（年均预算从2024年的120万美元 → 2026年的700万美元）
- Top 1%公司人均月AI支出7500美元（约5万人民币），与中位数公司差距达680倍

英国经济学家Jevons在1865年发现：当蒸汽机效率提升、煤炭消耗降低后，英国的煤炭总消耗量反而暴涨——因为便宜了，所以用更多。160年后，这个悖论在AI时代精确复现：token便宜了98%，但单次智能体调用从0.04美元涨到1.20美元（30倍），总用量暴增。

## 创作方向建议

### 主线一：Jevons悖论在AI时代的精确复现

**经典Jevons悖论回顾**：
1865年，英国经济学家威廉·杰文斯发现：瓦特改良蒸汽机后，每单位动力的煤炭消耗量大幅下降，但英国的煤炭总消耗量不降反升。因为效率提升→使用成本下降→使用场景暴增→总消耗暴涨。

**AI时代的复现**：
```
2022末：GPT-4级别 ≈ $20/百万token
2026年：GPT-4级别 ≈ $0.40/百万token  （降98%）

但：
2023年：简单线性流程每次交互 ≈ $0.04
2026年：编排良好的智能体系统每次交互 ≈ $1.20  （涨30倍）

企业AI预算：$120万/年 → $700万/年  （涨近5倍）
```

**为什么降了还贵了？**
- 智能体不是"问一个问题"，是编排多次调用+工具使用+自我纠错
- Claude Opus 4.5、GPT-5.1、Gemini 3 Pro显著放大了单任务token消耗
- 高盛预测：到2030年全球token使用量将增长24倍

> **创作建议**：用数据可视化呈现"单价降、总量涨、单任务涨"的三条曲线

### 主线二："可卡因成瘾"模型

**Priceline高管的原话**：
> "这就像可卡因成瘾。他们先让你免费试用让你上瘾，然后你就离不开了。"
> ——Chris Reed, Priceline IT财务高级总监

**成瘾机制拆解**：
1. **免费/低价获客**：Claude Max $20/月无限量、Cursor免费试用、各平台额度补贴
2. **深度绑定**：开发者工作流一旦嵌入AI，切换成本极高
3. **隐形涨价**：
   - Priceline一次常规Cursor续约涨价4-5倍
   - 有公司忘记设上限，单月跑了5亿美元Claude账单
   - Uber在4月就花光了全年AI编程预算
4. **戒断困难**：微软给开发者开了Claude Code，6个月后又收回——"反弹"惨烈

**数据佐证**：
- Ramp内部AI使用量同比增长6300%
- 99.5%员工使用AI工具，84%每周用编程Agent
- 每位开发者的token消耗，过去9个月增长18.6倍

> **创作建议**：不只写"成瘾"，也要写"成瘾后的理性"——企业开始设上限、做FinOps、质疑ROI

### 主线三：谁在花钱？Anthropic vs DeepSeek的竞争格局

**市场份额（Ramp数据）**：
