# Codex经济学：OpenAI首次公开Agent经济的真实数据

**日期**: 2026-06-26
**类型**: [数据报告]
**热度**: ⭐⭐⭐⭐

## 背景

2026年6月25日，OpenAI联合哥伦比亚大学、沃顿商学院、杜克大学发布了一份关于Codex经济影响的研究报告。这是AI Agent领域第一个由权威学术机构参与的经济量化研究。

### 核心数据

| 指标 | 数据 |
|------|------|
| 周活跃用户增长（2026 H1） | 5x |
| 非开发者用户增长 | 137x |
| 每日委派给Agent的任务等效≥8h人工的用户占比 | 25.6% |
| OpenAI Codex月活用户 | 突破千万级 |

### 关键洞察

这不是一份编程工具的增长报告——而是**Agent经济的第一次权威数据快照**。

- **137x非开发者增长**说明：Agent正在突破开发者圈层，进入企业普通员工的日常工作流
- **25.6%用户日委派8h+**：相当于四分之一用户每天把"一个人工"的工作量交给Agent——这是真正的生产力替代，不是效率提升
- **学术机构署名**（哥大、沃顿、杜克）：OpenAI在刻意将Codex的叙事从"技术故事"升级为"经济学故事"

### OpenAI Codex的身份转变

Codex从一个编程助手（Copilot升级版），正在变成**企业工作流操作系统**。它不再只是写代码的工具——而是在接管任务规划、跨系统协作、信息检索、文档生成等一整套知识工作链路。

如果这一趋势继续，Codex将不再跟GitHub Copilot比，而是跟Salesforce、SAP、Workday比——重新定义企业软件的市场格局。
