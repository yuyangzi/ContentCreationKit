# 中国科技巨头开源 AI 工具三连发：小米、华为、快手的一周密集输出

## 热度背景

### 事件概览
2026年6月中旬，三家中国头部科技企业在一周内密集发布/开源核心 AI 工具，分别精准卡位 AI 编程、具身智能、多模态视频三大赛道。

### 小米 MiMo Code（AI 编程智能体）
| 维度 | 详情 |
|------|------|
| 定位 | 基于 OpenCode 构建的终端 AI 编程智能体，主打长线自动化编程 |
| 协议 | **MIT 开源**，开发者可免费商用部署 |
| 核心能力 | 跨会话持久化项目记忆（SQLite FTS5）、多智能体协同架构 |
| 成本 | 完成 125 项开发任务、301 次 Git 提交、处理 3.87 亿 Token，总 API 成本仅 **$70** |
| 付费模式 | 入门版 $63/年、进阶版 $168/年 |

### 华为 CloudRobo（端到端具身智能开发平台）
| 维度 | 详情 |
|------|------|
| 定位 | 全球首个端到端具身智能开发平台 |
| 核心突破 | 打通数据喂养→模型训练→云端部署→系统集成全链路 |
| 创新点 | 数据与模型双重评估体系、模块化动力控模型积木式拼装 |
| 效率 | 机器人云端接入缩短至**小时级**，模型部署压缩至**分钟级** |
| 落地 | 国家地方共建人形机器人创新中心、亿嘉和、上海交大已完成验证 |

### 快手 Keye-VL-2.0（30B 多模态大模型）
| 维度 | 详情 |
|------|------|
| 定位 | 聚焦长视频理解与视频 Agent 开发场景 |
| 架构 | 30B MoE，运行时仅激活 **3B 参数** |
| 核心突破 | 深度稀疏注意力（DSA），**256K 超长上下文** |
| 能力 | 原生搭载代码编写、工具调用、网页搜索等 Agent 能力 |
| 开源 | Hugging Face + GitHub 双平台上线 |
