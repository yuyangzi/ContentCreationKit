# 中国超算"灵晟"登顶Top500：47000颗鲲鹏CPU、全栈国产化，纯CPU冲击E级性能

**日期**: 2026-06-24
**类型**: [趋势]
**热度**: ⭐⭐⭐⭐

## 背景

2026年6月，中国E级超算"灵晟"正式打榜Top500并登顶，成为全球唯一以**纯CPU架构**冲击E级性能门槛的超算系统。它不仅代表中国超算重回世界第一，更在技术路径选择上引发深度讨论。

### 核心参数

| 指标 | 数据 |
|------|------|
| 处理器 | 47,000颗华为鲲鹏CPU |
| 算力 | >2 ExaFLOPS |
| 部署单位 | 国家超级计算深圳中心 |
| 发布时间 | 2026年4月24日（Top500打榜为6月） |
| 国产化程度 | **全栈国产化，不含任何境外芯片** |

### 技术路径争议

- **纯CPU vs GPU加速**：全球主流E级超算均采用CPU+GPU混合架构（如Frontier、Fugaku），"灵晟"选择纯CPU路线
- **鲲鹏CPU能力**：ARM架构、自研核心，证明了国产CPU可以支撑E级计算
- **质疑声音**：纯CPU路线的能效比是否可持续？编程模型是否兼容现有超算软件生态？

### 象征意义

- **从"被制裁"到"登顶"**：在美国芯片禁令背景下，全栈国产化超算登顶具有强烈的叙事张力
- **华为的"第二曲线"**：鲲鹏CPU从服务器芯片走向超算核心，华为计算战略的延伸
- **中国超算路线的独立性**：不同于美日的CPU+GPU混合路线，中国走出了自己的技术路径
