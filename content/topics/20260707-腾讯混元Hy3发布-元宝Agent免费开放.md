# 腾讯混元 Hy3 发布：元宝 Agent 能力免费开放，国产大模型卷向 Agent 化

## 热度背景

2026 年 7 月 6 日，腾讯正式发布混元 Hy3 大模型，同步在元宝上线 Hy3 Agent 能力并**免费开放**。这是混元完成底层重建后的首个正式版本，采用 MoE 架构（2950 亿总参数 / 210 亿激活参数），支持 256K 上下文，以 Apache 2.0 协议开源。

**关键数据**：
- Hy3 preview 上线后，日均 Token 消耗量增长 **20 倍**
- 内部盲测中 Hy3 得分 2.67/4，超越 GLM 5.1（2.51/4）
- 元宝 Agent 任务完成率 93.7%，幻觉率下降超一半
- 已接入 WorkBuddy、CodeBuddy、Marvis、ima 等腾讯内部产品线
- 同步 day-0 接入 HuggingFace、ModelScope、OpenRouter 等海外平台

**行业背景**：国内大模型正从"卷参数"全面转向"卷 Agent"。字节豆包、阿里千问、智谱 GLM 等均在发力 Agent 能力，腾讯此次以"免费开放"姿态入场，价格战 + 生态战的组合拳值得关注。

## 类型标签

国产大模型 / Agent 生态 / 开源 / 行业竞争分析

## 创作方向

1. **产品对比向**：Hy3 vs GLM 5.1 vs 豆包/千问——国产 Agent 模型横评
2. **生态向**：腾讯的"模型+Agent+产品矩阵"打法分析（元宝/WorkBuddy/CodeBuddy/Marvis/ima）
3. **趋势向**：从"模型即服务"到"Agent 即服务"——国产大模型的商业化路径变迁
4. **开源向**：Apache 2.0 协议开源的策略意图——腾讯的开发者生态棋局
