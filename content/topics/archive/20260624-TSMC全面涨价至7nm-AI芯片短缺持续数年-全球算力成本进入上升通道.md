# TSMC全面涨价至7nm + AI芯片短缺"将持续数年"：全球算力供给侧巨变

**日期**: 2026-06-24
**类型**: [趋势]
**热度**: ⭐⭐⭐⭐⭐

## 背景

2026年6月，全球最大芯片代工厂台积电（TSMC）的两则消息震动了整个科技行业：一是对所有先进制程全面涨价，二是CEO魏哲家坦言AI芯片短缺"将持续数年"。这标志着AI时代的算力供给从"充足且价格下降"转向"持续紧缺且成本上升"的历史性拐点。

### TSMC涨价细节

据Culpium独家报道（6月23日），TSMC已通知客户以下变动：

- **涨价范围**：所有先进制程节点，包括7nm、5nm、3nm及以下
- **涨幅**：5%-10%（因节点和订单量而异）
- **波及面**：覆盖约75%的晶圆收入（Q1先进节点占晶圆收入75%）
- **意外之处**：连相对"老旧"的7nm也在涨价之列，部分客户感到震惊
- **已开始执行**：据消息源，涨价已启动

### TSMC市场地位（Q1 2026数据）

| 指标 | 数据 |
|------|------|
| Q1营收 | $359亿（+41% YoY） |
| 全球代工市场份额 | 72.3% |
| 与第二名（三星）营收比 | **11倍** |
| 先进制程（≤7nm）市占率 | >90% |
| 毛利率 | 66.2% |
| 全年资本支出 | $520亿-$560亿 |

### CEO魏哲家关键表态

- "AI芯片需求增长'疯狂'（insane）"
- "供给将滞后需求数年（for years），不是几个季度"
- 先进节点需求超产能约25-30%
- 先进节点产能已锁定至2027年以后
- "即使美国新建工厂也无法满足需求"

### TSMC vs 三星 vs Intel

- 三星代工Q1营收仅$32亿（TSMC的1/11）——史上最大差距
- Intel Foundry Q1营收$54亿，但外部客户仅$1.74亿（运营亏损$24亿）
- Intel虽拿下苹果部分订单（18A-P低端芯片），但TSMC仍握>90%先进芯片供应
- Intel股价年初至今涨260%，但技术差距仍以"年"计算

### 关联影响

- **AI推理成本上升** — 大模型公司面临的Token成本压力将进一步加剧
- **创业公司算力预算承压** — 小型AI公司的GPU/算力预算被挤压
- **终端产品涨价传导** — 从云端到终端的涨价链条
- **地缘政治叠加** — 美国对华芯片管制 + ASML EUV禁运 + TSMC涨价 = 三重供给侧压力

## 创作方向
