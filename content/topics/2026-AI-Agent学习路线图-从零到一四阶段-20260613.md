## 热度背景

### 现象概述

2026 年被业界广泛称为"Agent 井喷年"。技术焦点从"训练更大的基础模型"转向"构建更聪明的智能体应用"。学习资源也随之爆发，GitHub 上出现多星系统性教程，掘金等社区 Agent 相关文章频繁登顶热榜。

### 热度数据

- **掘金**：《2026 年从 0 开发 AI Agent 需要的 10 个技能》2007 热度（AI 榜 #1）
- **GitHub**：Hello-Agents（Datawhale）、ai-agents-from-zero、agent-craft 等教程获大量 Star
- **牛客网**：AI Agent 工程师学习路线讨论活跃
- **提效录**：2026 AI Agent 开发完全指南（从零到一）

### 行业背景

- 2025 年：AI Agent 元年（Manus 爆火、各大厂布局）
- 2026 年：Agent 从实验走向生产，企业开始规模化部署
- 框架生态巨变：厂商 SDK 强势崛起（Claude Agent SDK、OpenAI Agent SDK），LangChain 不再一家独大
- MCP 协议（Model Context Protocol）成为事实标准，被 OpenAI、Google、Cursor 全面采纳

### 学习路径已形成共识

正确的顺序（2026 版）：
1. **LLM 基础**（3-5 天）— 先搞懂 Agent 在工程上会坏在哪里
2. **Prompt Engineering + Function Calling**（1 周）— Agent 的核心引擎
3. **RAG + LangChain 基础**（1-2 周）— 搭好知识检索和工具调用
4. **LangGraph + MCP 协议**（2-3 周）— 状态管理、工具标准化
5. **多 Agent 协作 + 实战项目**（持续）— 企业级落地

常见误区（很多教程的错序）：**先学框架 → 搭 demo → 遇到真实场景就崩**。正确顺序应该反过来：先搞懂原理层面的坑，再学框架怎么填坑。

## 创作方向建议

### 核心叙事线

1. **引子（10%）**：2026 年为什么是 Agent 元年？从"百模大战"到"智能体井喷"的范式转移。都说要学 Agent，但大多数教程的顺序是错的
2. **一张全景路线图（10%）**：四阶段可视化总览——地基 → 手脚 → 协作 → 实战
3. **第一阶段：打好地基（20%）**：
   - LLM 调用原理（Transformer、MoE、推理 vs 训练）
   - Prompt Engineering（为什么它是 Agent 的基础？Chain-of-Thought、Few-shot 的底层逻辑）
   - Function Calling 机制——Agent 的"行动能力"是如何实现的？Tool Schema 设计原则
   - **为什么这个阶段不能跳过**：不理解 Function Calling 的底层机制，直接用 LangChain 就是在黑盒操作
4. **第二阶段：搭好手脚（20%）**：
   - RAG（检索增强生成）：Chunking 策略、向量检索 vs 关键词检索、Hybrid Search
   - LangChain 基础：六大模块（Models、Prompts、Chains、Memory、Indexes、Agents）
   - 构建第一个单 Agent：天气查询 + 数据库检索 + API 调用
5. **第三阶段：学会协作（20%）**：
   - LangGraph：图结构编排、状态管理、Human-in-the-loop
   - MCP 协议（2026 必学）：为什么是事实标准？Server/Client 架构、Tool/Resource/Prompt 三元组
   - 多 Agent 协作：CrewAI 入门、Agent 间通信、避免循环依赖
6. **第四阶段：实战打磨（15%）**：
   - 在 SWE-Bench Verified 上做 Code Agent（真实工程能力验证）
   - 企业级部署：流式输出、错误重试、安全防护、对话记忆管理
   - 性能优化三维度：精度 / 延迟 / 成本
7. **结尾（5%）**：Agent 开发是 2026 年最值得投入的技术方向，入门门槛已大幅降低。两周内完全可以上手

### 工具推荐

- **IDE**：Cursor、Claude Code（AI 辅助开发）
- **框架**：LangChain（入门）→ LangGraph（进阶）→ Claude Agent SDK（生产）
- **协议**：MCP（工具集成标准）
- **低代码/Demo**：Coze、Dify（快速验证想法）
- **模型**：Claude Fable 5（海外）、DeepSeek V4-Pro（国内）、Qwen 3.6（开源可用）
