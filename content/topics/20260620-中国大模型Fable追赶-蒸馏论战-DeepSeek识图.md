# 中国大模型追赶 Fable 级别：蒸馏论战、GLM-5.2 突破与 DeepSeek 识图

## 热度背景

### 什么是 Fable 级别？
Anthropic 于 2026年6月9日发布 Claude Fable 5，首个"Mythos-class"模型（Opus 之上新层级）。SWE-bench Verified 得分 95%，GPQA Diamond 约 95%，几乎所有基准 SOTA。发布仅3天（6月12日）被美国政府以国安为由强制下架。

### 马斯克 vs 唐杰隔空预测（6月18日）
| 预测者 | 预测时间 | 观点 |
|--------|---------|------|
| Elon Musk | Q1 2027 | 基准追赶容易，真实实用性才是考验 |
| 智谱 CEO 唐杰 | "不用那么久" | 暗示 Q4 2026，GLM-5.2 已接近 Opus 4.8 |

### GLM-5.2 硬核数据
- 744B 总参 / 40B 激活 MoE，MIT 开源
- **华为昇腾训练**，全程无 Nvidia，成本约 $2500 万
- CodeArena 全球可用模型**第一**，FrontierSWE 距 Opus 4.8 仅差 1%
- 发布后智谱股价单日涨 26.15%，市值超 9000 亿

### 蒸馏论战（6月15日）
Redis 之父 **antirez** 在 X 上公开反驳"中国模型靠 API 蒸馏"指控：
- 完整蒸馏需要 full logits + 中间输出，公开 API 只返回文本
- DeepSeek 已公开预训练/SFT/RL 管线，社区可复现
- 但 Anthropic 2月报告指控 DeepSeek 等通过 2.4 万欺诈账户 + 1600 万次 API 调用进行"蒸馏攻击"
- 争议本质：技术定义（白盒蒸馏 vs 黑盒 SFT）vs 舆论叙事的错位

### DeepSeek 识图全量上线（6月18日）
- 基于 V4-Flash MoE（284B总参/13B激活）
- "以视觉原语思考"——空间推理新范式
- 800×800 图片仅耗 ~90 tokens（Claude Sonnet 约870）
- 知乎单独话题 56 万热度

### 角度一："制裁加速国产替代"——美国封锁如何反向推动中国 AI

# DeepSeek V4思维链突变——V4.1要来了？

## 来源

- **知乎热榜**，246万热度，2026.06.16
- 链接：https://www.zhihu.com/question/2050272536898835791
- 新浪财经：DeepSeek V4登榜分析
- DeepSeek API更新日志

## 热度背景

知乎246万热度，AI技术圈最热话题之一。DeepSeek时隔15个月于2026.04.24发布V4，成为全球开源模型第2名。仅2个月后，用户发现思维链风格突变，引发V4.1即将发布的强烈猜测。同期"识图模式"灰度测试也在展开。

## 核心事件

### 思维链变化
- DeepSeek网页端/App"快速模式"思维链突然不再以"嗯，""好的，"开头
- 改为大量使用列表的结构化形式，风格偏向GLM/Qwen
- 多轮对话时更易触发，"专家模式"尚未变化
- "识图模式"灰度测试中也出现同样样式

### V4现状
- V4于2026.04.24发布：100万Token上下文、1.6T参数MoE架构
- V4 Pro定价：每百万token 24元人民币（约GPT-5.5的1/10）
- 支持华为昇腾950芯片部署

### V4.1传闻（计划2026.06上线）
- 新增多模态输入：图片、截图、图表、语音、录音理解
- 原生支持MCP（模型上下文协议）：直接对接API、数据库
- 强化企业级商用配套：API批量调用、模型微调、部署工具链

### DeepSeek × Kimi "开源双子星"
- 至少5次技术"撞车"：R1/K1.5推理模型、NSA/MoBA注意力架构、Prover数学推理、mHC/注意力残差架构创新、V4/K2.6万亿参数迭代
- 两家都支持国产芯片，都走开源路线，创始人都是广东人
- 中国开源AI阵营正在抱团突围

## 创作方向建议

- **角度A**：从思维链变化切入，猜测V4.1能力边界——多模态+MCP+企业级意味着什么？
- **角度B**：DeepSeek × Kimi"开源双子星"分析——中国AI的技术路线和生态战略
- **角度C**：定价屠夫——V4 Pro价格仅为GPT-5.5的1/10，开源模型的商业逻辑能跑通吗？
- **角度D**：华为昇腾生态——V4支持昇腾950，中国AI芯片能否突围？

## 核心背景

### 标志性转向

2026 年 2 月，OpenRouter 平台上**中国模型的 Token 消耗量首次超过美国模型**。到 Q2，中国提供商已占流量的 45%+。一年前这个数字不到 2%。

**小米一家流量就达到 OpenAI 的 3 倍。**

### 关键事件：GLM-5.2 发布（2026.06.13）

智谱发布了可能是 2026 年冲击力最大的中国模型：
- 753B 参数 MoE 架构
- 在多项编码基准上**击败 GPT-5.5**
- 价格仅为 GPT-5.5 的 **1/6**
- MIT 开源许可证
- 完全运行在华为昇腾芯片上——不受美国出口管制
- 智谱股价自年初涨了 600%+

### 价格战：中国比美国便宜 10-50 倍
