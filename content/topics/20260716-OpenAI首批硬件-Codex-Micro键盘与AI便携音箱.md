# OpenAI发布首批硬件：Codex Micro键盘与AI便携音箱

## 热度背景

2026年7月15日，OpenAI 正式发布其**首款品牌硬件产品 Codex Micro**——一款与机械键盘厂 Work Louder 联合设计的可编程宏键盘，售价 **$230**，面向使用 Codex AI 编程智能体的开发者。

这是 OpenAI 硬件野心的第一步。同期，Bloomberg 爆料 OpenAI 正在开发一款更激进的消费级设备：**屏幕无界面、便携式 AI 智能音箱**，集成了 ChatGPT，带有"可自主移动的机械部件"。该产品由前苹果首席设计官 **Jony Ive** 主导设计，预计2026年下半年发布。

值得注意的是，这两个硬件项目正面临 **Apple 的商业机密诉讼**——Apple 指控两名前苹果员工（现 OpenAI 员工 Tang Yew Tan 和 Chang Liu）窃取了硬件设计机密。

## Codex Micro 产品细节

- **定位**：为 Codex AI 编程智能体设计的专用物理控制器
- **合作方**：Work Louder（基于 Creator Micro 平台）
- **价格**：$230，限量发售
- **核心功能**：
  - 发光"Agent Keys"：显示智能体运行状态，一键切换多个智能体
  - 可编程 Command Keys：快捷触发常用操作（发送提示词、接受/拒绝代码变更、Push-to-Talk）
  - 摇杆：导航操作
  - 旋转拨盘：调节智能体的"推理深度"（reasoning level）
  - 32个可替换 Codex 主题键帽
  - 可选"clicky"或"silent"机械轴体
- 可通过 ChatGPT 桌面应用自定义所有按键

## AI智能音箱（传闻产品）

- 设计：Jony Ive 主导，无屏幕，便携式
- 特点：集成 ChatGPT，"可自主移动的机械部件"
- 状态：仍在开发中，细节可能变化
- 与 Codex Micro 无关，面向不同受众
