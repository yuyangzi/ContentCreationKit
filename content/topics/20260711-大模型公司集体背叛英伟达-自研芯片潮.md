# 大模型公司集体"背叛"英伟达：自研芯片潮

## 热度背景

2026年7月，36氪等多家媒体集中关注一个现象：全球所有头部大模型公司正在加速"去英伟达化"。从OpenAI到Anthropic，从DeepSeek到字节跳动，自研AI芯片成为标配。

**关键信号**：
- OpenAI自研芯片计划加速（Apple诉讼中暴露了其硬件野心）
- DeepSeek推理芯片已进入实质阶段（软硬一体极效路线）
- Anthropic在IPO文件中披露了自研芯片路线图
- Google TPU持续迭代，亚马逊Trainium/Inferentia规模化部署
- 微软、Meta也在推进自研ASIC
- 英伟达HBM最大供应商SK海力士美股上市，侧面印证AI芯片需求暴涨

**36氪标题**："为什么全球所有大模型公司，都开始'背叛'英伟达？"

**核心逻辑**：当模型推理成本成为AI商业化的最大瓶颈，依赖英伟达的通用GPU意味着将利润大头交给上游。自研芯片不是"要不要"的问题，而是"什么时候"的问题。

## 类型标签

`AI芯片` `产业竞争` `算力` `供应链`

## 创作方向建议

1. **成本算术**：自研芯片的ROI到底怎么算？以DeepSeek推理芯片为例，软硬一体的效率提升能否覆盖自研的巨额投入？
2. **路线分化**：不同公司的自研策略——DeepSeek的"推理极效"、OpenAI的"通用芯片"、Google的"云端闭环"、Meta的"开源硬件"。不是所有自研芯片都是同一件事。
3. **英伟达的护城河**：CUDA生态、NVLink互联、HBM供应——英伟达的壁垒不是芯片设计，而是整个生态。大模型公司能绕过几个？
4. **产业格局**：AI芯片从"一家独大"到"群雄逐鹿"，对算力成本、模型定价、AI应用生态的连锁影响。
