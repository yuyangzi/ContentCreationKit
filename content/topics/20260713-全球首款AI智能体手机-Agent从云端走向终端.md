# 全球首款 AI 智能体手机：Agent 从云端走向终端

## 热度背景

2026 年 7 月 7 日，上海市政府新闻发布会宣布，**全球首款 AI 智能体手机**将在 2026 世界人工智能大会（WAIC）首发首秀。该消息登上微博热搜，新浪财经、澎湃新闻、证券时报等多家媒体广泛报道。

**WAIC 2026 亮点**：
- 9 位图灵奖/诺贝尔奖得主参会，包括强化学习之父 Richard Sutton、深度学习三巨头之一 Yoshua Bengio
- 首发产品阵容：华为 Atlas 950 超节点、MiniMax M3 多模态大模型、阶跃 Agent 操作系统、近存计算 3D 芯片、**全球首款 AI 智能体手机**、多款人形机器人
- 大会聚焦世界模型、开源智能体、AI Coding、Token 经济、OPC 等前沿议题

**行业背景**：
- 2025 年底，字节跳动联合中兴努比亚推出"豆包 AI 手机" nubia M153，虽遭微信/淘宝等主流 App 限制，但持续与 vivo、联想、传音推进合作
- 智谱开源 AutoGLM——全球首个具备 Phone Use 能力的 AI Agent
- IDC 预计 2026 年中国新一代 AI 手机出货量达 1.47 亿台，同比增长 31.6%，占整体市场 53%
- 专家观点：2026 年大概率成为 **AI 手机元年**，但总体进展不会像预期那样快

**核心悬念**：谁在做这款"全球首款 AI 智能体手机"？目前信息未明确透露厂商。可能的候选包括华为、小米或某家新锐厂商。AI 手机的核心命题是：Agent 能否真正替代 App 成为人机交互的新入口？

## 2026-07-13 更新

**努比亚确认发布全球首款 AI Agent 手机**。ZTE 高级副总裁倪飞在努比亚官方微博确认，该机将在 WAIC 2026（7月17-20日，上海）公开展示。

**核心能力**：搭载字节跳动豆包 AI，运行在操作系统层面，使用 GUI Agent 读屏技术——AI 可自主打开 App、跨 App 比价、填写表单、完成下单，全程无需用户触碰屏幕。

**关键背景**：
- 前代 nubia M153（豆包 AI 手机）3 万台一天售罄，但微信、淘宝等主流 App 几乎立即封杀其自动化操作
- 新机策略不明：是继续"硬刚"App 生态，还是走 API 合作路线？三星和苹果的 Agent 方案倾向于 API 模式，努比亚的 GUI Agent 路线更激进
- Counterpoint Research 预测 2026 年 GenAI 手机占全球出货量 45%（2025 年为 36%），三星在 Agentic AI 功能上暂时领先
- 无西方发布计划，字节跳动的 TikTok 监管摩擦使其出海困难

**核心悬念升级**：从"谁在做"变为"怎么活"——前代被 App 生态封杀，这次努比亚能否突破？GUI Agent vs API Agent 两种路线，谁会成为主流？
