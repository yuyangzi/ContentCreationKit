# DeepSeek $74亿首轮融资内幕：梁文锋自掏200亿控盘，商业投资者锁仓5年无权

**日期**: 2026-06-24
**类型**: [热点]
**热度**: ⭐⭐⭐⭐⭐

## 背景

2026年6月中旬，DeepSeek完成了成立以来的首轮外部融资，募集总额超500亿人民币（约$74亿）——这不仅是中国AI行业史上最大单轮融资，其独特的交易结构更是引发全球关注。

### 核心数据

- **募资总额**：500亿+人民币（$74亿）
- **投后估值**：约4000亿人民币（$590亿），中国最贵AI初创公司
- **超越对象**：Moonshot AI（$300亿估值）、MiniMax（HK$1242亿市值）
- **仍落后于**：智谱AI（HK$4890亿市值 / $950亿估值）

### 交易结构（教科书级的控制权设计）

| 投资者 | 金额 | 权利 |
|--------|------|------|
| **梁文锋（创始人/CEO）** | 200亿元 | 通过有限合伙结构管理，保持绝对控制 |
| **腾讯** | 100亿元 | 5年锁仓，无投票权 |
| **宁德时代（CATL）** | 50亿元 | 5年锁仓，无投票权 |
| **国家AI产业投资基金** | 未透露 | **有投票权，无锁仓限制** |
| 其他（网易、京东、IDG等） | 未透露 | 5年锁仓，无投票权 |

### 关键结构特征

1. **投资者不直接持有DeepSeek股份**，而是通过梁文锋管理的有限合伙结构注资
2. **所有商业投资者锁仓5年**，无投票权——防止早期退出和干预公司方向
3. **国家基金例外**：持有投票权且无锁仓——政府获得有效治理权
4. **梁文锋个人出资占比约40%**（200亿/500亿），远超任何单一机构投资者

### 梁文锋的态度

据SCMP报道，梁文锋在上月投资者线上会议中多次重申：
- "DeepSeek想做的是任何与提升AI模型智能相关的事"
- "对其他事情毫无兴趣"
- 坚持AGI路线，不被商业化压力左右

### 背后的"量化基金养AI"模式

- 梁文锋同时是幻方量化（High-Flyer Quant）联合创始人
- 幻方管理资产超700亿，2025年回报56.6%，年收入超$7亿
- 全部反哺DeepSeek研发——这种"量化基金养AGI研究"的模式全球独一无二
- 此次外部融资标志着这一模式的转折点
