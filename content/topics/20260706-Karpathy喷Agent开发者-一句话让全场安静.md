# Karpathy喷Agent开发者：一句话让全场安静

## 热度背景

2026年7月初，据36Kr引述报道，Andrej Karpathy在一次AI行业大会上公开批评Agent开发者："你们做的Agent根本不可靠。"

作为前Tesla AI总监、OpenAI创始成员，Karpathy在AI开发者社区拥有极高声望。他的一句话让现场陷入沉默，随后在中文技术社区引发热议。

Karpathy的核心批评指向当前Agent开发的浮躁状态：大量公司在没有解决可靠性问题的情况下就急于商业化Agent产品，把不成熟的系统包装成"下一代AI应用"。他的批评与近期Agent落地僵局、豆包千问关停智能体等事件形成共振——行业开始从Agent狂热中冷静下来。

在掘金社区，"别被模型宣传骗了，真实 Agent 任务一跑就知道"等文章从工程角度验证了Karpathy的观点：模型在benchmark上表现优异，但在真实Agent任务中经常翻车。
