# 审核主题

## 目标

对已选定的创作主题进行深度审核，确保方向明确、细节清晰、背景资料充分。

## 前置条件

已完成 `/find-popular-topics`，或已有确认的创作主题

## 输入

- `content/topics/` 目录下的主题文件，或对话中已确认的主题

## 步骤

1. 使用 `grill-me` skill 对主题内容进行反复拷问和细节确认，一轮只问一个关键问题，直到没有任何模糊点。
2. 使用 `recursive-research` skill 和搜索类 MCP 工具，深度收集与主题相关的背景信息。
3. 将确认的问答记录和收集的信息整理到 `content/reference/` 目录，文件名为 `{时间戳}-{主题名}.md`。
4. 如有问题，给出修正意见供我审核，不要直接修改主题文件。

## 输出

- `content/reference/{时间戳}-{主题名}.md`：包含审核问答记录和背景研究资料

## 约束

**必须遵守：**

- 一条一条提问，每轮只问一个关键问题
- 引用信息须提供来源，不得捏造

**禁止操作：**

- 不要一次性抛出多个问题
- 不要直接修改主题文件内容
