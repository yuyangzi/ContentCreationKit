# 对用户提供的内容创作主题进行深度审核，步骤如下

1. 结合上下文信息使用 `grill-me` skill对主题内容进行反复拷问和细节确认，直到一切细节确认清楚且没有任务模糊点。

2. 使用 `recursive-research` skill 和 相关搜索类的MCP工具，深度收集一切与主题相关的信息。

3. 将确认好的问题和用户回答，以及搜集到的信息，全部都进行整理并以主题文件同名的文件名存在到当前`content\reference` 文件夹内。

4. 如有存在任何问题给出修正意见供我审核，不要直接修改。
