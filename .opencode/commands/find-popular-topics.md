# 查找热门主题

## 目标

在各大内容平台上挖掘当前高热度的话题，结合用户指定的方向筛选出适合创作的候选主题。

## 前置条件

无

## 输入

- （可选）用户指定的创作方向或领域

## 步骤

1. 派遣 `librarian` Agent使用MCP工具，在内容平台搜索当前高热话题。如有指定方向，优先搜索该方向；无指定方向则搜索综合热度最高的话题。尽可能的在搜寻更多的平台或网站。
2. 收集最多 10 个话题，再派遣 `librarian` Agent搜索每个话题的背景信息（热度趋势、受众关注点、已有内容的覆盖程度）。
3. 获取到话题和背景信息后使用 `brainstorming` skill 与我逐一讨论，筛选出最适合创作的主题。一次只讨论一个主题，逐个确认。
4. 将确认的主题信息以 Markdown 文件保存到 `content/topics/` 目录，文件名为 `{时间戳}-{主题名}.md`。

## 输出

- `content/topics/{时间戳}-{主题名}.md`：包含主题名称、热度背景分析、创作方向建议

## 约束

**必须遵守：**

- 话题来源须可回溯，优先抓取原文页面内容而非摘要
- 与用户讨论时一次只讨论一个主题

**禁止操作：**

- 不要编造话题热度数据
- 不要一次性抛出多个话题让用户选择
