# 生图提示词

## 目标

根据文章内容，生成适用于 AI 绘图工具的微信文章封面提示词。

## 前置条件

已完成 `/to-article`，或已有确认的正式文章

## 输入

- `content/article/` 目录下的文章文件，或用户提供的文章内容

## 步骤

1. 精读文章内容，提取核心主题、关键意象、情感基调。
2. 分析文章的目标受众和发布平台，确定封面风格方向。
3. 生成 2-3 组不同风格的中文生图提示词。
4. 每组提示词附带对应的画面描述说明。

## 输出

- 2-3 组生图提示词（在对话中呈现），每组包含：
  - 风格标签（如：写实摄影 / 矢量插画 / 3D 渲染）
  - 完整的中文 prompt
  - 画面预期说明（一句话描述最终视觉效果）

## 约束

**必须遵守：**

- 提示词与文章内容高度相关，不跑题
- 提供中文 prompt（主流 AI 绘图工具的最佳语言）
- 每组风格有明显差异化

**禁止操作：**

- 不要生成与文章调性不符的风格（如严肃科技文用卡通可爱风）
- 不要忽略平台的封面图尺寸要求（如有）
