# 设计文档：AI压缩执行力 HTML PPT

## 项目概述

将PPT文案转化为可录屏的HTML演示文稿，用于B站视频制作。

| 属性 | 值 |
|------|-----|
| 用途 | B站视频录屏素材 |
| 幻灯片数 | 25-35张（约31张） |
| 风格 | 深色极简 |
| 交互 | 键盘翻页 + 鼠标点击（纯手动，无自动播放） |
| 动画 | 简单淡入淡出 |
| 文件 | 单HTML文件，CDN加载Reveal.js |

---

## 技术选型

### 框架：Reveal.js
- 成熟的演示框架，45k+ stars
- 内置键盘/触摸/URL导航
- 支持淡入淡出过渡
- CDN加载（`https://cdn.jsdelivr.net/npm/reveal.js@5.1.0/`）

### 样式：内联CSS
- CSS变量控制主题

### 图表：纯CSS + SVG
- 柱状图、卡片对比、进度条用CSS实现
- 左右分栏对比图、before/after柱对比用CSS flexbox
- 分叉流程图用CSS + 箭头字符实现
- 无外部图表库依赖

## 交互规范

### 翻页方式

```javascript
// 键盘
Reveal.configure({
  keyboard: true,        // 方向键翻页
  keyboardCondition: 'focused'
});

// 鼠标
Reveal.configure({
  mouseWheel: true,      // 滚轮翻页
  fragments: true        // 点击显示分步内容
});
```

### 过渡动画

```javascript
Reveal.initialize({
  transition: 'fade',       // 淡入淡出
  transitionSpeed: 'slow',  // 慢速过渡
  progress: false           // 隐藏进度条（录屏时UI噪音）
});
```

### URL导航

支持URL参数控制：
- `#/slide/5` — 跳转到第5张

## 已确认决策

### 技术决策

| 问题 | 决策 | 备注 |
|------|------|------|
| 加载方式 | CDN加载 | Reveal.js 5.1.0 jsDelivr CDN，需要网络 |
| 翻页方式 | 纯手动键盘+鼠标 | 方向键翻页、滚轮翻页，彻底移除自动播放 |
| 录屏安全区 | 不需要 | 无B站播放器遮挡考虑，铺满全屏 |
| 章节标题页 | 普通编号标题 | 居中显示，非全屏分隔页 |
| 进度指示器 | 隐藏 | Reveal.js 自带进度条在录屏时属于UI噪音 |
| 全屏按钮 | 保留 | 右上角全屏按钮（Fullscreen API） |
| 显示优化 | v1一并实现 | CSS铺满窗口、图表自适应、气泡比例修复 → 不另开v2 |
| 目标环境 | Chrome @ 1920×1080 macOS | 录屏和测试均以此为基准分辨率 |
| 字体方案 | 系统字体栈 | `PingFang SC / Microsoft YaHei`，不加载思源黑体 |
| 幻灯片方向 | 全部水平 | 31张均为顶层 `<section>`，禁止嵌套垂直子幻灯片 |
| CSS约束 | 锁定 `!important` 范围 | 仅4处已声明规则可用 `!important`，禁止新增 |
