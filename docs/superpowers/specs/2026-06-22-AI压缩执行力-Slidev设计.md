# 设计文档：AI压缩执行力（Slidev + neocarbon）

## 项目概述

将文章《AI压缩了执行力，放大了判断力》转为 Slidev HTML 演示文稿，用于 B站 视频录制。

| 属性 | 值 |
|------|-----|
| 用途 | B站视频录屏素材 |
| 幻灯片数 | ~31 张 |
| 目标观众 | 泛科技爱好者 |
| 故事风格 | 数据驱动 + 叙事 + 观点输出 |
| 框架 | Slidev 52.0.0 + `@enyineer/slidev-theme-neocarbon` 1.0.8 |
| 动画策略 | 保留全部（staggered entrances、shimmer、particles） |
| 翻页 | 键盘纯手动 |
| 比例 | 1920×1080 (16:9) |
| 输出 | `slidev build` → `dist/` SPA |

## 配色方案

在 `slides.md` 的 `<style>` 块中设置：

```css
:root {
  --nc-accent:  #ff6b35;   /* 橙色 → 中性强调、关键数据 */
  --nc-success: #22c55e;   /* 绿色 → 正面/上升/默会知识 */
  --nc-danger:  #ef4444;   /* 红色 → 负面/下降/警示 */
  --nc-warning: #f59e0b;   /* 琥珀 → 保留 */
  --nc-info:    #3b82f6;   /* 蓝色 → 保留 */
}
```

### 颜色编码规则

| 数据类型 | 颜色 | 实现 |
|----------|------|------|
| 增长/正面/默会知识 | 绿色 | `nc-text-success` 或 `var(--nc-success)` |
| 下降/负面 | 红色 | `nc-text-danger` 或 `var(--nc-danger)` |
| 中性/强调 | 橙色 | `nc-text-accent` 或 `var(--nc-accent)` |

**严格禁止**：红色用于中性数据；默会知识/价值判断用橙色代替绿色。
